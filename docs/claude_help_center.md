# Claude Help Center Reference

## Troubleshooting Message Errors
Claude users can troubleshoot message errors by checking network status, request size, and model availability.
If an API request fails, users should review status messages and retry with exponential backoff.
Rate limit errors can be resolved by reducing request frequency and batching smaller prompts.

## Workspace and Permissions
Workspace role changes, permissions updates, and admin seat management must be escalated to human support.
If a user loses workspace access after an IT admin removes their seat, they should contact their workspace admin or escalate to Claude support.
Account recovery, identity verification, and security incidents must be handled by human support specialists.

## Billing
Billing disputes and invoice corrections are handled only through billing support specialists.
Claude billing questions should be directed to the billing team and are not handled through general support.

## API Usage
Claude API users experiencing errors should check the Anthropic status page and review their API key permissions.
If an API key is invalid or expired, users should regenerate it from the Anthropic console.
